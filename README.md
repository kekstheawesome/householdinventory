# Household Inventory

Full-stack shared household inventory application for a single household with one admin and one or more roommates.

## Stack
- Backend: ASP.NET Core Web API, Clean Architecture, EF Core, SQL Server, Identity, JWT
- Frontend: React, TypeScript, Vite
- AI: Gemini through backend-only integration

## Projects
- `src/backend/HouseholdInventory.Api`
- `src/backend/HouseholdInventory.Application`
- `src/backend/HouseholdInventory.Domain`
- `src/backend/HouseholdInventory.Infrastructure`
- `src/frontend`

See `docs/SETUP.md` for setup instructions.
